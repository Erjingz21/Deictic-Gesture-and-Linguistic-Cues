# Deictic Gesture and Linguistic Cues

Reproducibility materials for:

> Zhang, E., Neer, E. M., & Sandhofer, C. M. *How caregivers coordinate deictic gestures with referential and syntactic structure.* (Manuscript under review.)

This repository contains the de-identified utterance-level data and the R script needed to reproduce the analyses, tables, and figures reported in the manuscript. Study materials (coding manuals, lexicon example, reliability documentation) are hosted on the project's [OSF page](https://osf.io/ka9gu/). Source video recordings are available to authorized researchers through Databrary.

## Project summary

We analyzed 21,220 caregiver utterances from home recordings of 15 American, English-speaking infants (22–23 months) to examine how caregivers coordinate deictic gestures with referential and syntactic structure. Each utterance was coded for the presence of nouns and pronouns, mean length of utterance (MLU), the presence of a deictic gesture (showing or pointing), and — for mixed noun-pronoun utterances — whether the noun and pronoun referred to the same entity.

## Data dictionary

`data/socialcue_data.csv` contains one row per coded caregiver utterance.

| Column | Type | Description |
|---|---|---|
| `ID` | integer | De-identified child identifier |
| `MLU` | integer | Mean Length of Utterance, calculated in morphemes |
| `Showing` | 0/1 | 1 if the utterance was accompanied by a showing gesture |
| `Pointing` | 0/1 | 1 if the utterance was accompanied by a pointing gesture |
| `Noun_count` | integer | Number of nouns in the utterance |
| `Pronoun_count` | integer | Number of pronouns in the utterance |
| `Same_Referent` | 0/1 or NA | For utterances containing both a noun and a pronoun, 1 if they refer to the same entity, 0 otherwise; NA when not applicable |

Empty cells in `Showing`, `Pointing`, `Noun_count`, and `Pronoun_count` are treated as 0 in the analysis script. The script also derives:

- `Social_Cue` — binary gesture indicator combining `Showing` and `Pointing`
- `utterance_type` — categorical (Only Pronoun, Only Noun, Pronoun & Noun, No Pronoun or Noun)
- `MLU_z` — `MLU` standardized to *z*-scores (mean 0, SD 1); regression models in Q3 use the standardized version so coefficients are interpretable per 1-SD change.

## How to reproduce

1. Clone or download this repository.
2. Open `analysis.R` (an RStudio project at the repo root works well, since the script uses `here::here()` for paths).
3. Install the required packages (R ≥ 4.2 recommended):

   ```r
   install.packages(c(
     "here", "readr", "dplyr", "tidyr",
     "lme4", "car", "broom.mixed",
     "emmeans", "ggeffects", "ggplot2"
   ))
   ```

4. Run the script:

   ```r
   source("analysis.R")
   ```

   Tables are written to `tables/` as CSV. Figures are written to `figures/` as PNG. Model summaries and tests are printed to the console.

The script is organized to mirror the four research questions in the manuscript: (Q1) gesture probability by utterance type, (Q2) referential density and the noun × pronoun interaction, (Q3) syntactic complexity (MLU) and its interaction with utterance type, and (Q4) same-referent vs. different-referent gesture use within mixed utterances.
